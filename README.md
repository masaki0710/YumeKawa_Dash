# YumeKawa Dash

研究室のUnity制作課題で作成。
プレイヤーの移動・ジャンプ操作で障害物を避ける耐久ゲーム。

## 操作方法

- W: 前
- S: 後
- A: 左
- D: 右
- Space: Jump
  - 長押しでジャンプ力上昇
